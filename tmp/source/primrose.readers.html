primrose.readers.r_reader module¶

Module with AbstractReader implementation, able to read datasets from R

This is a simple example to show that primrose can work with other languages. Note:

It works with simple 2D R dataframes (e.g. iris dataset) but not higher tensors (e.g. iris3 dataset) It does not work with timeseries, such as lh dataset

We perform an R as.data.frame conversion on all data to help stadardize the data structure. For instance, the R euro dataset is essentially a labelled vector but converts nicely to a dataframe.

Author(s):

Carl Anderson (carl.anderson@ww.com)
